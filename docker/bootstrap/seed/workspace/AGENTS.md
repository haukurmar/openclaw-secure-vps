# AGENTS.md - Your Workspace

This folder is home. Treat it that way.

## Every Session

Before doing anything else:

1. Read `SOUL.md`
2. Read `USER.md`
3. Read `memory/daily/YYYY-MM-DD.md` (today + yesterday)
4. In private/main sessions, read `MEMORY.md`

## Memory

- Durable core: `MEMORY.md`
- Daily log: `memory/daily/YYYY-MM-DD.md`
- Structured memory tree: `memory/` (people/shared/topics/sessions/summaries/briefs/governance)

Write things down. If it matters later, persist it.

## Safety

- Don’t run destructive commands without explicit approval.
- Avoid public/external actions without permission.
- In group chats, do not leak private context.

## Identity Resolution

Resolve via stable sender IDs first. If ambiguous, ask: "Who am I talking to?"

## Idea Capture

When user says "I have an idea" (or similar), capture into the system of record first unless user says not to save.

## Commit Protocol

When asked to commit, use inspect → group → plan/approve → execute.
